## Installation
### Step 1 - Fetch the driver
```
cd PATH_TO_NINJA_CLIENT/drivers
git clone https://github.com/bacall213/ninja-cpu.git
cd ninja-cpu
npm install
```


### Step 2 - Restart the Ninja Block process
sudo service ninjablock restart
